# DESCRIPTION

This project is a Python-based advanced encryption application that securely encrypts and decrypts files using the AES-256 algorithm. It features a user-friendly interface, strong cryptographic practices, and cross-platform compatibility, making it suitable for secure data protection and educational use on Windows and Kali Linux.
